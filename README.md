## ***Watch - Получение данных на основе местоположения пользователя***

Приложение "Watch" позволяет получить данные: страну, город, время, дату и др.дополнительную информацию по местоположению пользователя. Выполнена адаптивная вёрстка (320px - 1440px). Использовано публичное API. Собственный дизайн приложения.
